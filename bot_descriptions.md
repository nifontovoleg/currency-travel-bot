# 🤖 Описания для Telegram-бота

Готовые тексты для настройки бота через [@BotFather](https://t.me/BotFather).

## Short description

**Лимит:** 120 символов

```
Миникошелёк для путешествий: конвертируй расходы, веди баланс в двух валютах и следи за тратами в поездке.
```

---

## Description

**Лимит:** 512 символов

```
✈️ Travel Wallet — ваш миникошелёк в поездке.

Создайте путешествие, укажите валютную пару и стартовый баланс — бот будет конвертировать каждый расход в домашнюю валюту.

💸 Отправьте сумму числом — бот пересчитает и предложит учесть трату.
💰 Баланс всегда в двух валютах.
🗺️ Несколько путешествий — переключайтесь одной кнопкой.
📋 История расходов и смена курса.

Команды: /start /newtrip /balance /history /setrate
```

---

## About

**Лимит:** 120 символов

```
Кошелёк путешественника: расходы, курсы, баланс в двух валютах. Просто отправьте сумму числом.
```

---

## Команды для BotFather

```
start - Главное меню
newtrip - Создать новое путешествие
switch - Выбрать активное путешествие
balance - Показать баланс
history - История расходов
setrate - Изменить курс обмена
```
